Creepy Chocolate Maze

Winner of Trijam #108: The 3 hour game jam.

This is a horror maze game with invisible walls, where you have to find your way in the maze to get a chocolate. After eating a chocolate, a new one is generated at another position and a new maze is generated, then you have to repeat the process and eat the next chocolate.

Game made in Love2D in only 3 hours. I've used the Aldous-Broder algorithm to generate the mazes and it worked like a charm, it was a nice programming challenge.

This is a really fun game that you can't stop playing. The atmosphere provided by the music, screen shake and game feel is very creepy and engaging.

DownUp

Game made in 4h for Trijam #80.

The theme of the jam was: "What goes up (must come down)". I explored the theme by making a game where you control the gravity and has to navigate your way between the platforms to make sure your character stays on screen while the game screen scrolls up automatically. Reach the 100th floor to win the game! Inspired by VVVVVV and Icy Tower.

Game made with the Phaser game engine and written in Javascript. It was my first time using Phaser so it was quite the challenge to learn it while making a game in such little time.
